# ElectriciansNearYou Landing Page

A modern, conversion-optimized lead-generation landing page for local electricians.

## 🚀 Features

- **GitHub Pages ready:** Just upload static files.
- **Mobile-first, responsive** (Tailwind CSS)
- **SEO & Open Graph meta** for Google and social previews
- **Fast, animated, premium UI**
- **Animated stats, testimonials, coverage, FAQ accordion**
- **Modern accessibility & conversion practices**
- **Schema.org `LocalBusiness` rich snippet**
- **Floating call button + sticky quote CTA**

## 🛠 Deployment

1. **Fork or clone this repository**  
2. **Upload files to the `main` branch** (or the default branch)
3. **Enable GitHub Pages**:
   - Go to your repo → `Settings` → `Pages`
   - Under "Source" select your default branch (`main` or `master`)
   - Root directory (`/`)
4. Access your live site at:  
   `https://<your-username>.github.io/<repo-name>/`

## File Structure

- `index.html` : Main landing page
- `script.js` : Animations, mobile nav, FAQ, contact handler
- `README.md` : This file

## 🌐 Customize

- Update company contacts, locations, testimonials in `index.html`
- Swap hero and team images for local ones (royalty-free Unsplash/placeholder used)
- Replace social and legal links
